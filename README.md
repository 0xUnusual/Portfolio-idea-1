# Junior Montero | Software Developer Portfolio 🚀

¡Bienvenido a mi portfolio personal! Soy un apasionado del desarrollo de software enfocado en crear experiencias web excepcionales, modernas y de alto rendimiento. Como graduado del **ITLA**, me especializo en construir soluciones full-stack robustas.

## ✨ Características Principales

- **Diseño Dark Luxury**: Estética moderna con glassmorphism y efectos visuales de alta gama.
- **Optimización Extrema**: Aceleración por hardware (GPU), throttling de eventos y carga diferida para un scroll fluido (60 FPS).
- **Responsive 0.1%**: Adaptabilidad total desde dispositivos móviles hasta pantallas Ultra-Wide.
- **Tech Stack Dinámico**: Banner animado con iconos de tecnologías dominadas.
- **Efectos Interactivos**: Tarjetas con efectos 3D (Vanilla-tilt.js) y animaciones de entrada (ScrollReveal).
- **Funcionalidad Real**:
  - Descarga directa de CV.
  - Formulario de contacto integrado con **WhatsApp**.
  - Enlaces directos a perfiles profesionales.

## 🛠️ Tecnologías Utilizadas

- **Core**: HTML5 Semántico, CSS3 Vanilla, JavaScript Moderno (ES6+).
- **Librerías**:
  - [Typed.js](https://mattboldt.github.io/typed.js/) - Animación de texto.
  - [ScrollReveal.js](https://scrollrevealjs.org/) - Animaciones de entrada.
  - [Vanilla-tilt.js](https://micku7zu.github.io/vanilla-tilt.js/) - Efectos 3D.
  - [Grommet Icons](https://icons.grommet.io/) - Iconografía técnica.
- **Infraestructura**: Git, GitHub Pages, Docker (Ready).

## 🚀 Despliegue

La web está optimizada para ser desplegada en **GitHub Pages**, Vercel o Netlify. 

1. Clona el repositorio:
   ```bash
   git clone https://github.com/0xUnusual/Portfolio-idea-1.git
   ```
2. Abre `index.html` en tu navegador o usa una extensión de Live Server.
